Variables and Usage: [length type]
* np 	: no. parameters [1 integer]
* pars 	: starting parameters [np double]
* ofn 	: obj function [1 function]
* efn 	: equality function [1 function]
* ifn 	: inequality function [1 function]
* neq 	: no. equalities [1 integer]
* nineq	: no. inequalities [1 integer]
* tc	: no. of total constraints  = neq + nineq [1 integer]
* fv 	: function value [1 double]
* ev 	: equality constraint value [neq double]
* iv 	: inequality constraint value [nineq double]
* feiv 	: vector of fv, ev, iv [(1+neq+nineq) double]
* pb	: inequality and bound constraints c(ineqLB, ineqUB, LB, UB) [(nineq*2 + np*2) integer]
* lpb	: signal switch : [2 integer]
				[1]=1 then par bounds exist
				[2]=1 the either par bounds or inequality bounds exist
* ctrl	: solver control vector [9 integer&double]
				[1] rho 	: interior penalty parameter [1 double]
				[2] maxit 	: max inner (subnp) interations [1 integer]
				[3] delta	: relative step size in forward difference evaluation [1 double]
				[4] tol		: tolerance on feasibility and optimality [1 double]
* g		: gradient of inequality and objective function [nineq+1 double]
* gap	: lower and upper "gap" [2 double]
* alp	: alpha value [3 double]
* ch	: change [1 double]
* scale	: scale vector
* h		: hessian [np*np double]
* tt	: vector of computed tolerances during optimization [3 double]
				[1] is the difference of the objective values between two consecutive iterations
				[2] is norm(constraint) before a major iteration
				[3] is norm(constraint) after a major iteration
